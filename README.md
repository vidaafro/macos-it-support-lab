# macOS IT Support Lab

## Overview
This project demonstrates foundational macOS IT support skills used in workplace technology and help desk environments. The lab focuses on device setup, troubleshooting, software installation, system settings, and technical documentation.

## Skills Demonstrated
- macOS configuration
- User account management
- Software installation
- Wi-Fi troubleshooting
- Technical documentation
- Help desk support workflows
- System maintenance

## Tools Used
- macOS
- Zoom
- Slack
- Google Workspace
- Microsoft Office
- System Settings

## Project Tasks
- Configured macOS settings for a simulated employee setup
- Installed and verified workplace software applications
- Reviewed security and privacy settings
- Troubleshot common connectivity and application issues
- Created technical documentation and support ticket examples

## What I Learned
This project helped strengthen my understanding of macOS device support, technical troubleshooting, and IT documentation practices commonly used in enterprise support environments.
