# macOS Device Setup Checklist

## Initial Setup
- Connect device to Wi-Fi
- Verify macOS is updated
- Configure display and accessibility settings
- Verify storage and battery health

## User Configuration
- Create or verify user account
- Configure password settings
- Set up iCloud and browser preferences

## Software Installation
- Install Google Chrome
- Install Zoom
- Install Slack
- Verify applications launch properly

## Security Checks
- Review Privacy & Security settings
- Verify firewall settings
- Confirm software updates are enabled

## Final Verification
- Test internet connection
- Restart device
- Verify applications function correctly
