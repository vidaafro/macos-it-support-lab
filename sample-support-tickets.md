# Sample macOS Support Tickets

## Ticket 001 - Wi-Fi Connectivity Issue
Issue:
User reported inability to connect to office Wi-Fi network.

Troubleshooting Steps:
- Restarted Wi-Fi connection
- Forgot and reconnected to network
- Restarted device
- Verified internet connectivity

Resolution:
Device successfully reconnected to network.

Status:
Resolved

## Ticket 002 - Zoom Microphone Permissions
Issue:

Troubleshooting Steps:

- Enabled microphone permissions for Zoom
- Restarted Zoom application
- Tested microphone input

Resolution:
Microphone functionality restored successfully.

Status:
Resolved
