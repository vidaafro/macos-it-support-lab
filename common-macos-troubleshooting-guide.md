# Common macOS Troubleshooting Guide

## Wi-Fi Connectivity Issues
1. Verify Wi-Fi is enabled
2. Restart the device
3. Forget and reconnect to the network
4. Verify internet access
5. Check for macOS updates

## Application Not Responding
1. Force quit the application
2. Restart the Mac
3. Verify software is updated
4. Reinstall application if needed

## Low Storage Space
1. Open Storage Settings
2. Remove unnecessary downloads
3. Empty Trash
4. Restart device

## Audio or Microphone Issues
1. Verify microphone permissions
2. Check input/output settings
3. Restart Zoom or application
4. Test microphone functionality

## Slow System Performance
1. Close unnecessary background applications
2. Restart the device
3. Review Activity Monitor
4. Install system updates
